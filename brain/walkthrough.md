# Walkthrough

## Proje Kurulumu

1. Repoyu clone edin
2. `index.html` dosyasını tarayıcıda açın (sunucu gerekmez)

## Önemli Dosyalar

- `index.html` - Tüm uygulama (HTML + CSS + JS)
- `README.md` - Proje dokümantasyonu
- `brain/` - Geliştirici notları ve planlar

## Kod Yapısı

### HTML

- Canvas container ve kontrol panelleri
- Tailwind CSS ile responsive tasarım

### JavaScript (Inline)

- Grid/Manuel mod yönetimi
- Canvas çizim ve event handling
- ZIP export (JSZip)
- Animasyon oynatıcı

## Notlar

- Tüm işlemler client-side yapılır
- Görsel veriler sunucuya gönderilmez
- Modern tarayıcılarda çalışır (ES6+)
